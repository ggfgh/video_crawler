# video_crawler
## 功能简介
* 爱情岛论坛视频爬虫，实现自动化下载指定id的视频
* 视频的清晰度一般，可能会有一些小问题

## 使用说明
* 建议python版本
python >= 3.7
* 安装依赖项
pip install -r requirement.txt
