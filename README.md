# video_crawler
## 功能简介
* 一个视频爬虫，实现自动化下载指定id的视频
* 视频的清晰度由爬取的原视频清晰度决定

## 使用说明
* 建议python版本>= 3.7
* 安装依赖项
pip install -r requirement.txt

```
-i -id          指定视频最后的id, 如16034
-t --threads    爬取的线程数，默认20
Example: python video_crawler.py -i 16034
```
